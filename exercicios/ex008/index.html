Principais formatações

Negrito / Destaque

Nesta frase temos um termo em negrito usando a tag <b>

Nesta frase temos um em destaque usando <strong>

Itálico / Ênfase

Nesta frase temos um termo em itálico usando <i>

Nesta frase temos um termo em itálico usando <em>

Texto marcado

Nesta frase temos um termo marcado usando <mark>

Nesta frase temos um termo marcado com a cor verde usando <mark style = "background-color">

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno neste parágrafo usando <big> e <small>

Texto deletado

Podemos marcar um texto como excluido para indicar que ele deve ser lido, mas não considerado.Fazemos isso usando a tag <del>

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois. Fazemos isso usando a tag <ins>

Existe também o sublinhado com a tag <u> (que é uma tag não-semântica)

Texto sobrescrito

Para inserir coisas do tipo x^20+3, formatando da forma x20+3, usando a tag <sup>

Texto subscrito

Para inserir coisas do tipo H2O, formatando da forma H2O, usando a tag <sub>
